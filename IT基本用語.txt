HTML／CSS／JavaScriptから構成される最も有名なWEBフレームワーク
フォーム、ボタン、メニューなどの部品のテンプレート

Flyway
オープンソースのデータベースマイグレーションツール
データベースの状態をバージョン管理できるようになる

Jackson
javaのライブラリで、JSONオブジェクトをJavaオブジェクトにマップしたりその逆を行うことができる
JSON・・・JavaScript Object Notationの略→テキストベースのデータフォーマット

jQuery
JavaScriptでできることを、より簡単な記法で実現できように設計されたJavaScriptライブラリ

Junit
Javaで開発されたプログラムにおいてユニットテスト（単体テスト）の自動化を行うためのフレームワーク

Log4j
Jakarta プロジェクトで開発が進められている Java プログラム用のログ API

Lombok
JavaのボイラープレートコードをシンプルにしてくれるJavaのライブラリ

ORM(Object-relational mapping)
・データベースからデータを取得する
・取得したデータをオブジェクト化する
・データの更新・変更などをデータベースに格納する

■ 連絡ツール
ChatWork
Chatwork株式会社が提供する、ビジネスコミュニケーションに特化したチャットツール

Slack
アメリカ発のコミュニケーションツール 
絵文字のカスタマイズや既存サービスとの連携できる

IP Messenger
 LAN 用メッセンジャーソフト
同じネットワーク内のメンバーを自動認識し、簡単にクライアント同士がリアルタイムに情報交換できる

■ Java
Javadoc
Javaのソースコードを元に自動生成されたソースコードの説明書

アノテーション
データやプログラムの中に特殊な記法を用いて埋め込まれた付加情報

メモリ管理
コンピュータのメモリを管理する。メモリはOSが監理している
プログラムの中で必要になるとメモリを借りる

ヒープ
コンピュータのメモリ領域の一部

GC（ガベージコレクション）
プログラムで使っていないメモリを解放する機能
GoF(Gang of Four)
オブジェクト指向における再利用のためのデザインパターン
デザインパターン
JavaやRubyなどのオブジェクト指向の言語で使われる設計パターンのこと
過去のエンジニアが解決してきた方法がまとめられている。
